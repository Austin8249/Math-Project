  • Twelve Basic Functions

    1. The Identity Function f ( x ) = x
    2. The Squaring Function f ( x ) = x 2
    3. The Cubing Function f ( x ) = x 3
    4. The Reciprocal Function f ( x ) = 1 x
    5. The Square Root Function f ( x ) = x
    6. The Exponential Growth Function f ( x ) = e x
    7. The Natural Logarithm Function f ( x ) = ln x
    8. The Sine Function f ( x ) = sin ( x )
    9. The Cosine Function f ( x ) = cos ( x )
    10. The Absolute Value Function f ( x ) = | x | = abs ( x )
    11. The Greatest Integer Function f ( x ) = int ( x )
    12. The Logistic Function f ( x ) = 1 1 + e - x
