Graphical Transformations

y = a * f ( b ( x - c ) ) + d

f = Parent Function


When a is greater than 1 the Function Stretches on its Y-value
When a is between 0 and 1 the Function Shrinks on its Y-value
When a is less than 0 the Function reflects on its X-Axis (y=0)
When b is greater than 1 the Function Shrinks on its X-value
When b is between 0 and 1 the Function Stretches on its X-value
When b is less than 0 the Function Stretch on its X-value
When c is positive the Function moves Left
When c is negative the Function moves Right
When d is positive the Function moves Up
When d is negative the Function moves Down

Example:

f ( x ) = - 4 ( - 2 x - 2 ) + 1

The Parent Function f ( x ) = x is:
  1. Reflected on the X-Axis
  2. Y-Value is stretched by 4
  3. Reflectd on the Y-Axis
  4. X-Value is shrinked by 2
  5. Right 2
  6. Up 1
